C++ cast

  1. static_cast
  2. when implicit casting is possible

    int x = static_cast<int>(3.14);

  3. when convertors are available Box b = static_cast<Box>("10,20,30");
  4. related down-cast or up-cast Base * bp = static_cast<Base*>(new Derived());
  5. inverse of implicit conversion int * np = static_cast< int* >( void_ptr_x );
  6. enum to int int one = static_cast<int>( ENUM_NUMBERS );
  7. int to enum ENUM_NUMBER n = static_cast<ENUM_NUMBERS>( one );

  8. enum to another enum

    ARABIC_NUMBERS an = static_cast< ARABIC_NUMBERS >( enumbers );

  9. dynamic_cast

  10. only on pointers or references
  11. for polymorphic types
  12. specially used for downcast

    Derived *pd = dynamic_cast< Derived* >( basePtr );

  13. does run-time check if down-cast succeeds if( Derived * pd = dynamic_cast < Derived* >( basePtr ) ) { /// success, can use pd }

  14. can be used to add 'const'ness

  15. constcast

  16. cast away the constness or volatility int h = 10; const int &o = h; const_cast<int &>(o) = 20;
    • works on pointers and references only
  17. destination-type also has to be of same type

  18. BUT, if the value itself is 'const' then result is undefined const int ci = 4; int * pi = const_cast< int* >( &ci ); // cannot use pi to modify value, undefined behavior

  19. reinterpretcast

  20. only for pointers and references
  21. from any type to any other type of pointer
  22. reinterprets types from the bit pattern int i = 10; char* p = reinterpret_cast< char* >( i );
