# git commands

> `git config -global user.name "first-name last-name"`

> `git config -global user.email "<email-address">"`

> `git init <repo-name>`

> `git clone <url>`

> `git branch [--all] <branch-name>`

> `git checkout <branch-name>`

> `git pull --rebase`

> `git rebase origin/master`

> `git cherry-pic <commit-id>`

> `git add .`

> `git add <filename/s>`

> `git commit -m "<msg>"`

> `git commit -am "<msg>"`

> `git diff [-staged]`

> `git diff <branch-1> <branch-2>`

> `git reset [-hard] <file>`

> `git reset <commit-id>`

> `git status`

> `git rm <file/s>`

> `git mv <src-file> <dest-file>`

> `git log [-p|--patch] -n <number>`

> `git log --oneline --graph`

> `git shortlog -nse --author=<email-address>`

> `git shortlog -nse --author=<username> --since="1 year ago"`


sows metadata of a commit
> `git show <commit-id>` 

> `git tag <commit-id>`

> `git branch -d <branch-name>`

> `git stash`

> `git stash pop`

> `git stash apply`
