Time Complexity in Big-O Notation

Time complexity is denoted using Big-O notation

Index Name of Time Complexity (TC) Rating Big-O Notation Examples
1 Constant TC BEST O( 1 ) Push, Pop, Index based returning
2 Logerithmic TC Better O( log n ) Binary search
3 Linear TC Good O( n ) Sum of all elements in 1D array, Linear search, Compare pair of strings
4 Linearithmic TC OK O( n log n ) Merge sort, Quick sort
5 Quadratic TC NOT GOOD O( n ^ 2 ) Sum of all elements in 2D array (matrix), Bubble sort, traversing 2D array
6 Polinomial Array TC BAD O( n ^ k ) Cubic, n^3 or n^4
7 Exponential TC WORST O( k ^ n ) Traveling Salesman
