Big-O Notation

Time complexity is denoted using Big-O notation

Index Graph Name of Time Complexity (TC) Rating Big-O Notation Examples
1 TODO Constant TC BEST O(1) Push, Pop, Index based returning
2 TODO Logerithmic TC Better O(logn) Binary search
3 TODO Linear TC Good O(n) Sum of all elements in 1D array, Linear search, Compare pair of strings
4 TODO Linearithmic TC OK O(nlogn) Merge sort, Quick sort
5 TODO Quadratic TC NOT GOOD O(n^2) Sum of all elements in 2D array (matrix), Bubble sort, traversing 2D array
6 TODO Polinomial Array TC BAD O(n^k) Cubic, n^3 or n^4
7 TODO Exponential TC WORST O(k^n) Traveling Salesman
