Data Structures (DS)

DS is methods of organizing data for effective use

Array

  • Attributes
    • Has size
    • Contiguous, Sequencial
    • Of same type
    • Easy to find Position
    • Index based
  • C++
    • array[index]
    • STL: std::array
  • Note
    • Bound checking is important

Stack

Attributes: Last In, First Out (LIFO) or First In, Last Out (FILO) Insertion and Deletion from one end only. Maintains reference to top of the stack. Push, Pop operations Check Empty or (occasionally, Check Full) operations
C++ STL: std::stack
Note, Stack unwinding concept, Stack overflow concepts

Queue

Attributes: First In, First Out (FIFO), First come first serve Insertion from one end, removal from other end. Maintains front and back of the queue. C++ STL: std::queue Operations: front, back, push at end, pop from front, empty Implementation: Deque, Priority Queue, Circular Queue
